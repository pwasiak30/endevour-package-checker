🔎 EOS Pkg Checker

Wpisz nazwę pakietu, aby sprawdzić jego najnowszą wersję w repozytoriach, z których korzysta EndeavourOS (Arch: core/extra/multilib + AUR), a następnie porównać z Debian, Ubuntu (wszystkie wydania, w tym LTS), Fedora, openSUSE, Gentoo i Linux Mint.

Projekt jest w 100% statyczny (GitHub Pages) — dane są wyliczane cyklicznie przez GitHub Actions i zapisane jako pliki JSON w repo, żadnego backendu na żywo. Wyszukiwarka działa tylko dla pakietów z listy poniżej (ładowanie…).
Źródła danych: Arch Linux Packages, AUR, Repology.org (Debian, Ubuntu, Fedora, openSUSE, Gentoo). Wersja dla Linux Mint to estymacja z bazowego wydania Ubuntu (Repology nie śledzi Mint osobno). Dane odświeża codziennie GitHub Actions — tam też można ręcznie doraźnie dodać/odświeżyć konkretny pakiet (przycisk „Run workflow”, pole „package”). Porównanie wersji jest przybliżone — różne dystrybucje stosują różne konwencje numeracji (epoch, sufiksy dystrybucyjne, patche).
